Welcome to GrilleLib Studio

Seamless AG Grid Community Custom Filtering

GrilleLib provides premium open-source custom filters that bridge the functional gaps between AG Grid Community and Enterprise. It allows corporate and indie-hacker consumers to integrate enterprise-grade set filters, regex validators, advanced calendars, and even Gemini-powered AI column filters into their own front-end projects, completely for free under the GNU AGPL-3.0 Copyleft License.

Avoid paying $1,200/year for enterprise licences. Get searchable, sorting multi-checkbox filters seamlessly on AG Grid Community.

psychology AI Semantic Queries

gavel 100% Copyleft Guarantee

# 1. Install required community packages
npm install ag-grid-community ag-grid-angular --save
